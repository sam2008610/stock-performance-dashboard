# 📈 個人投資記錄追蹤器

一個簡單實用的個人投資記錄管理工具，幫助您追蹤投資交易和計算基本績效。

## ✨ 主要功能

### 📝 交易記錄管理
- **交易記錄**：記錄買入/賣出交易，包含日期、數量、價格等基本資訊
- **多資產類型**：支援台股、美股、加密貨幣、債券、金融商品的分類標記
- **基本搜尋**：提供台股代碼搜尋功能
- **資料儲存**：使用瀏覽器本地儲存保存交易記錄

### 📊 基本統計分析
- **持股總覽**：顯示持有股票清單和基本數量統計
- **成本計算**：計算平均成本和總投入金額
- **簡單圖表**：提供基本的圓餅圖和面積圖視覺化
- **損益概覽**：顯示基本的損益統計（需手動更新股價）

### 💾 資料管理
- **本地儲存**：所有資料保存在瀏覽器本地，不上傳到伺服器
- **基本匯出**：支援交易記錄的基本匯出功能
- **快取管理**：提供清除快取的選項
- **資產歷史**：記錄資產變化的基本歷史

### 🎨 介面設計
- **響應式設計**：適配桌面和手機瀏覽
- **深色/淺色主題**：可切換主題模式
- **簡潔介面**：專注於實用性的簡單設計

## 🚀 技術實現

### 前端技術
- **Vue 3** + **Nuxt 3**：現代化前端框架
- **Tailwind CSS**：CSS 框架
- **Chart.js**：圖表功能
- **LocalStorage**：本地資料儲存

### 資料來源
- **FinMind API**：台股基本資料（僅用於搜尋驗證）
- **手動輸入**：股價需要手動輸入和更新
- **本地資料庫**：內建台股清單用於搜尋

## 📦 安裝與使用

### 系統需求
- Node.js 16.0 或更高版本
- 現代瀏覽器（支援 LocalStorage）

### 安裝步驟

1. **下載專案**
```bash
git clone https://github.com/your-username/stock-tracker.git
cd stock-tracker
```

2. **安裝相依套件**
```bash
npm install
```

3. **啟動開發伺服器**
```bash
npm run dev
```

4. **開啟瀏覽器**
```
http://localhost:3000
```

### 部署
```bash
npm run build
npm run preview
```

## 📖 使用說明

### 1. 初始設定
1. 首次使用需要設定目前現金餘額
2. 設定開始追蹤的日期
3. 系統會根據交易記錄反推歷史資產變化

### 2. 記錄交易
1. 點擊「新增交易」
2. 選擇交易類型（買入/賣出）和資產類型
3. 輸入股票代碼、數量、價格等資訊
4. 確認後會自動更新持股記錄

### 3. 查看統計
- **投資組合頁面**：查看持股清單和基本統計
- **交易記錄頁面**：瀏覽所有交易歷史
- **圖表顯示**：查看資產分布和變化趨勢

### 4. 資料管理
- **設定頁面**：調整現金餘額、重建資產歷史
- **清除快取**：重新載入股票資料
- **重置功能**：清除所有記錄重新開始

## ⚠️ 重要說明

### 功能限制
- **股價更新**：股價需要手動輸入，不會自動更新
- **資料同步**：資料僅保存在本地瀏覽器，不會同步到其他裝置
- **計算精度**：損益計算基於輸入的價格，可能與實際市價有差異
- **資料備份**：建議定期匯出資料進行備份

### 適用對象
- 個人投資記錄管理
- 基本的投資組合追蹤
- 學習投資記錄的初學者
- 不需要複雜分析功能的使用者

## 🔧 API 說明

### 內建 API
```
GET /api/stock-list        # 取得台股清單
GET /api/stock?symbol=xxx  # 查詢股票基本資料
POST /api/clear-cache      # 清除伺服器快取
```
